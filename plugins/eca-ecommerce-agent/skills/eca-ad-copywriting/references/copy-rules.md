# Copy Rules — the mechanical brief

These are the exact, non-negotiable rules for producing the deliverables. When you present copy, it must obey these. Count characters before presenting — the limits are hard rules.

---

## 1. Headlines — write 10

**Hard rules**
- **Maximum 45 characters, including spaces and emojis.** This is the single most important rule. Short wins on Meta.
- **Do NOT start a headline with an emoji.** Emoji may appear mid- or end-of-line where it earns its place — used strategically, not on every line.
- Each headline must **describe what the person expects to see when they land on the page** — the product and its core benefit. The ad and the page must share the same "scent" so the click converts.

**Each of the 10 should**
- Target a different audience segment (spread them across the segments given).
- Tap a core human desire/emotion (status, ease, belonging, health, time, confidence, savings, indulgence…).
- Align with the campaign's stated business objective.

**Quality bar**
- High curiosity, scroll-stopping, benefit-forward.
- Use real customer language where you have it (from reviews/personas).
- No vague brand fluff — say something concrete a stranger would stop for.

**After presenting:** ask the user to pick their **2 favourites** (or request refinements). Wait.

---

## 2. Primary text — write 2 versions

### Version 1 — Short
- **Maximum ~120 characters**, including spaces and emoji.
- Punchy. Leads with the **main benefit** and how the product **solves/transforms the main pain point**.
- **No emoji at the start of a sentence.**
- Brand tone of voice.
- **No CTA** in this version.

### Version 2 — Long
- **First line: an emotion-led hook of ≈100–140 characters** that taps the most relevant human desire (highlight the main benefit through emotion).
- **No emoji at the start of a sentence.**
- Then the body:
  - **4 short bullet points**, each highlighting a **benefit (not a feature)**.
  - **Do not** use the phrase "why you'll love it".
  - **Each bullet starts with an emoji.**
  - Use line spacing between bullets for readability.
- Then a **compelling call-to-action with an irresistible offer**, optionally appending the landing-page URL.

**Refine** Version 1 and Version 2 until the user is completely satisfied. Wait between rounds.

---

## 3. The 3-2-2 method — package for testing

Once copy is approved, lay it out for the media buyer using the **3-2-2 method**:

- **3 creatives** — three variations of the **same ad angle**. Example: one UGC problem/solution concept, shot/edited with **3 different hooks** (so the angle is held constant and only the opening varies).
- **2 headlines** — the two the user chose.
- **2 primary texts** — Version 1 (short) and Version 2 (long).

This gives a clean, controlled test: same angle, varied hooks, two headlines, two body copies. Remind the user that the 3 creatives should stay within one angle so the test reads cleanly — don't mix angles inside a single 3-2-2 set.

---

## Pre-flight checklist before presenting any copy
- [ ] Every headline ≤ 45 characters (counted, incl. spaces/emoji)
- [ ] No headline or sentence starts with an emoji
- [ ] Version 1 ≤ ~120 characters and has no CTA
- [ ] Version 2 opening line ≈100–140 characters, 4 emoji-led benefit bullets, ends on CTA + offer
- [ ] Bullets are benefits, not features; no "why you'll love it"
- [ ] Claims are ones the brand can actually make
- [ ] Customer language used where available
