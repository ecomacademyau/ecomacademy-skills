# SEO + AI Audit — a Shopify visibility skill

Its one job: **raise a Shopify store's visibility in search engines and AI answer engines** — get found in Google/Bing *and* cited by ChatGPT, Perplexity, Claude and AI Overviews. Ships as **`eca-seo-ai-audit`** in the ECA Marketing Coworker. Built to run **monthly**.

## What it does
1. **Connects the required tools** — Shopify, Google Search Console, Bing Webmaster, DataForSEO, and Claude in Chrome. It helps connect any that are missing and **won't run a half-blind audit without them**.
2. **Works out the keywords** — the best terms to rank for and how the store already appears (GSC + Bing + DataForSEO).
3. **Audits** technical, on-page, and off-page (+ AI answer-engine readiness).
4. **Scores it 0–100** in a **simple, non-technical, non-alarmist summary** — where you're strong, where the quick wins are.
5. **Walks you through the fixes** as an easy **Now / Later / Ongoing** plan, quick wins first.
6. **Implements on approval** — makes the changes for you where it can (content via Shopify, theme via a dev theme), always shown and approved first, then re-scores.

## Principles
- **Plain English, never alarmist** — written for a store owner, not an SEO.
- **Quick wins first** — highest visibility gain for least effort.
- **Visibility ≠ capture** — the goal is getting *found* (indexed, ranked, AI-cited); CTR/clicks is a separate traffic thing, not scored as SEO.
- **Propose → approve → apply** — never edits the live theme directly; real data only.
